# Super Trunfo de Países (em C)

Trabalho simples em linguagem C que simula o jogo Super Trunfo, com cartas de países.

## 📌 Atributos das cartas
- Código da carta  
- País  
- Capital  
- Continente/Estado  
- População  
- Área (km²)  
- PIB  
- Pontos turísticos  

## 🚀 Como executar
Compile com:
```bash
gcc main.c -o supertrunfo
./supertrunfo
